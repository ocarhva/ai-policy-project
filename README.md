# Detecting Bias Signals in Hospitality Reviews using Python & NLP

This project explores whether hospitality reviews may contain potential bias signals in language, ratings, and location-based patterns.

The analysis uses:
- Python
- Pandas
- NLP sentiment analysis
- Data visualization

The goal is not to automatically prove discrimination, but to build an explainable first-pass analysis that can help teams investigate fairness risks responsibly.

## Skills Demonstrated
- Data analysis
- Natural Language Processing (NLP)
- Ethical AI thinking
- Data visualization
- Python for policy analysis
